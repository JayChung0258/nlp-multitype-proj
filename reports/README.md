# Reports Directory

This directory contains generated reports from EDA and analysis notebooks.

## Generated Reports

### EDA Report
- **File:** `eda_report.md`
- **Generated by:** `notebooks/00_eda.ipynb`
- **Contents:** Comprehensive summary of dataset statistics, validation results, and training recommendations

To regenerate:
```bash
cd /path/to/nlp-multitype-proj
source venv/bin/activate
jupyter notebook notebooks/00_eda.ipynb
# Run all cells
```

The report will be automatically saved to `reports/eda_report.md`.
